# Linux Setup

### Update
```bash
sudo apt-get update
```

### user as sudo
```bash
sudo usermod -a -G sudo $USER
```

### Tools
```bash
sudo apt install curl
```

```bash
sudo snap install node --channel=8/stable --classic

npm install --global yarn@1.13.0
```
